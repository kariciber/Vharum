# VHARUM

Pequeño experimento visual / interactivo en HTML5.

El proyecto carga imágenes directamente desde el repositorio y renderiza una escena simple en el navegador.  
Actualmente funciona como una demo base para pruebas de concepto.

## Estado del proyecto

En desarrollo 🚧

Características actuales:

- Renderizado en canvas
- Carga de imágenes desde GitHub (RAW)
- Escena básica
- Personaje visible en pantalla
- Movimiento simple

## Controles

Teclado:

- **W A S D** o **Flechas** → mover
- (según versión) **E** → interactuar

En móvil:

- Compatible con navegador moderno
